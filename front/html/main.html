Добавить рецепт Войти Регистрация
Наше приложение - это база с рецептами и видео по готовке. Вы можете произвести поиск необходимого блюда: по ингредиентам, сложности и длительности готовки. Понравившиеся рецепты можно сохранять и просматривать на странице профиля. Так же вы можете загрузить свой рецепт на сайт и его будут видеть другие пользователи. Внутри каждого рецепта можно можно просчитать необходимое количество ингредиентов для определённого количества порций
